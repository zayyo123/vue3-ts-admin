## 基于 Vue3+TypeScript 的后台管理系统

> ### 1.线上体验

[Vue3+TS 后台管理系统在线体验](http://cms..ltd)

**客官如果觉得还不戳，github 上不妨顺手点个 :star:，鼓励一下吧。**

> ### 2.项目运行

- git clone 到本地
- cd 到 vue3_ts_cms 目录下运行：

```
npm install 或者 cnpm install
```

- 运行

```
npm run serve
```

- 浏览器打开 http://localost:8080/
